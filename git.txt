1- git init
Initializes a new Git repository in the current directory.

2- git clone <repository-url>
Clones a remote repository to your local machine.
Example: git clone https://github.com/user/repo.git

3- git add <file>
Adds the specified file(s) to the staging area for the next commit.
Example: git add filename.txt

4- git commit -m "commit message"
Records changes to the repository with a descriptive message.
Example: git commit -m "Initial commit"

5- git status
Displays the state of the working directory and staging area (files modified, staged for commit, etc.).

6- git log
Shows the commit history of the repository.

7- git branch
Lists, creates, or deletes branches.
Example: git branch (lists branches), git branch new-branch (creates a branch)

8- git checkout <branch-name>
Switches to the specified branch.
Example: git checkout feature-branch

9- git merge <branch-name>
Merges the specified branch into the current branch.
Example: git merge feature-branch

10- git push
Uploads local repository changes to a remote repository.
Example: git push origin main
